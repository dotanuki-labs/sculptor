## What

This project is intended to be used within Dotanuki Labs only.

## Using

- Get this project
- Rename it
- Run the `sculpt.sh` script

```bash
$> git clone git@github.com:dotanuki-labs/sculptor.git
$> mv sculpt my-tool
$> cd my-tool
$> ./sculpt.sh my-tool
```

- Setup git and start a new project on Github
