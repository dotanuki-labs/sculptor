# sculptor

## What

An extremely opinionated way to bootstrap CLI apps powered by Rust.

## Using

- Get this project
- Rename it
- Run the `sculpt.sh` script

```bash
$> git clone git@github.com:dotanuki-labs/sculptor.git
$> mv sculpt my-tool
$> cd my-tool
$> ./sculpt.sh my-tool
```

- Setup git and start a new project on Github
